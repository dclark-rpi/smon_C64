I found this example on Kevin's Blog at 
https://emalliab.wordpress.com/2026/01/02/rp6502-the-picocomputer/
it is based on Rumbledethumps example on his youtube channel. The article can be read
at the address above.
I've found that Kevin's example doesn't end gracefully and requires a ctrl + alt + del
key combination to return to the commandline. 

I've fixed that problem in the assembler source below and I also added in a new byte
sequence to make sure it works nicely with smon and converted it for the C64.

You can type the object code into smon you just type the code below at the command prompt
including the ':'
You just type G 0200 and press return to execute the code.


Assembler Source:

* = $0200
ldx #0
loop:
lda text,x
jsr $ffd2
inx
cmp #0
bne loop
jmp $a474

text:
.BYTE $0D $0A
.ASCII "Hello, World!"
.BYTE $0D $0A $00

Object Code:

:0200 A2 00 BD 10 02 20 D2 FF
:0208 E8 C9 00 D0 F5 4C 74 A4
:0210 0D 0A 48 45 4C 4C 4F 2C
:0218 20 57 4F 52 4C 44 21 0D
:0220 0A 00
G 0200
